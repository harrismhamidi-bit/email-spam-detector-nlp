# Email Spam Detector NLP

This project is an NLP-based Streamlit web application that detects whether an email message is spam or legitimate.

## Theme

Theme 4: Email Spam Detector

## Project Objectives

- To classify email messages as spam or ham.
- To apply NLP preprocessing techniques.
- To compare machine learning models.
- To build an interactive Streamlit web application.
- To visualize dataset insights and model performance.

## Features

- Email text input
- Spam or ham prediction
- Confidence score
- Dataset preview
- Word cloud visualization
- Class distribution chart
- Confusion matrix
- Model comparison chart
